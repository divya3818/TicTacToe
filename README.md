# Tic Tac Toe 🎮

A simple, interactive classic Tic Tac Toe game built with HTML, CSS, and Vanilla JavaScript.

## Demo 🚀

[Live Demo](https://tic-tac-toe-flax-delta.vercel.app/)

## Features ✨

- **Two-Player Gameplay:** Take turns playing as Player O and Player X.
- **Win Detection:** Automatically detects horizontal, vertical, and diagonal winning combinations.
- **Move Validation:** Prevents players from overwriting previously played boxes.
- **Game Controls:** Includes "Reset Game" and "New Game" buttons to easily clear the board and start over.
- **Dynamic UI:** Displays a celebratory message congratulating the winner when the game ends.

## Technologies Used 🛠️

- **HTML5:** For structuring the game board and user interface elements.
- **CSS3:** For styling the grid, buttons, and responsive design.
- **JavaScript (ES6):** For the core game logic, DOM manipulation, and event handling.

## How to Play 🕹️

1. Clone the repository to your local machine:
   ```bash
   git clone <your-repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd TicTacToe
   ```
3. Open the `index.html` file in your favorite web browser.
4. **Player O** starts first. Click on any empty box on the grid to place your marker.
5. Players alternate turns until someone aligns 3 of their marks in a row (horizontally, vertically, or diagonally).
6. A message will pop up announcing the winner.
7. Click **Reset** or **New Game** to clear the board and play again.

## Project Structure 📁

- `index.html`: The main markup for the game.
- `style.css`: Contains the visual styling for the game board and UI.
- `app.js`: Handles the game logic, state, and interaction.
